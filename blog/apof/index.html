Aplicaciones Ofimáticas

SOBRE EL MÓDULO

¿Qué es la ofimática?

Es el conjunto de herramientas que usamos para trabajar en oficina: procesadores de texto, hojas de cálculo, presentaciones, etc.

Programas comunes

  • Word / Writer: para escribir documentos.
  • Excel / Calc: para hacer cálculos y gráficos.
  • PowerPoint / Impress: para presentaciones.

Funciones básicas

  • Crear, guardar y abrir documentos.
  • Formatear texto (negrita, cursiva, subrayado).
  • Crear tablas, listas, insertar imágenes.
  • Usar fórmulas y funciones básicas en hojas de cálculo.
