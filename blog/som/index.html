Sistemas operativos monopuestos

SOBRE EL MÓDULO

¿Qué es un SO monopuesto?

Un sistema que se instala en un solo ordenador y es usado por un usuario a la vez.

Ejemplos

  • Windows 10
  • Ubuntu Desktop
  • macOS

Tareas comunes

  • Instalación y configuración
  • Creación de usuarios
  • Instalar software
  • Gestión de permisos y carpetas

Comandos Linux útiles

  • ls, cd, mkdir, sudo, apt install
