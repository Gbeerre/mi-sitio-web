Redes

SOBRE EL MÓDULO

¿Qué es una red?

Una red es un conjunto de dispositivos conectados entre sí para compartir recursos e información.

Tipos de redes

  • LAN (Local Area Network): Red local, por ejemplo, una red de casa o de clase.
  • WAN (Wide Area Network): Red más grande, como Internet.
  • MAN (Metropolitan Area Network): Red a nivel de ciudad.

Dispositivos de red

  • Router: Conecta redes diferentes.
  • Switch: Conecta varios dispositivos dentro de una red.
  • Access Point (AP): Permite conexión WiFi.

Direccionamiento IP

  • Cada dispositivo tiene una dirección IP única.
  • IPv4: 192.168.1.1 (más común)
  • IPv6: formato más largo y moderno.

Modelo OSI (7 capas)

  1. Física
  2. Enlace de datos
  3. Red
  4. Transporte
  5. Sesión
  6. Presentación
  7. Aplicación

Cableado

  • UTP (par trenzado): el más usado.
  • Fibra óptica: más rápida y costosa.

Comandos útiles

  • ping: Ver si un equipo responde.
  • ipconfig o ifconfig: Ver la IP.
  • tracert / traceroute: Ver la ruta de paquetes.
