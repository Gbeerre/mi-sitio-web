Aplicaciones Web

SOBRE EL MÓDULO

¿Qué es una aplicación web?

Una app web es un programa que funciona desde el navegador, como Google Docs o Gmail.

Frontend vs Backend

  • Frontend: lo que ve el usuario (HTML, CSS, JavaScript).
  • Backend: lo que ocurre en el servidor (bases de datos, lógica).

Tecnologías básicas

  • HTML: estructura de la página.
  • CSS: estilo.
  • JavaScript: interacción.

Herramientas útiles

  • Visual Studio Code
  • Navegadores como Firefox o Chrome (consola de desarrollador)
